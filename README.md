# 🏥 Sistema Clínico

## 📌 Descripción
El **Sistema Clínico** es una aplicación web desarrollada en **C#**, **ASP.NET** y **CSS**, diseñada para la gestión integral de pacientes, doctores y personal administrativo dentro de una clínica.  
Su objetivo es optimizar procesos internos, mejorar la atención médica y centralizar la información en un sistema seguro y escalable.

---

## 🚀 Características principales
- **Gestión de pacientes**: registro, historial clínico y seguimiento.
- **Acceso por roles**:
  - Administradores (`admin_access`)
  - Doctores (`doctor_access`)
- **Módulos informativos**: páginas como `About.aspx` para información institucional.
- **Interfaz web responsiva** con estilos personalizados en `Styles/`.
- **Scripts dinámicos** para mejorar la experiencia del usuario.
- **Base de datos integrada** en `App_Data` para almacenamiento seguro.

---

## 🛠️ Tecnologías utilizadas
- **Lenguajes principales**:
  - C# (58.1%)
  - ASP.NET (35.8%)
  - CSS (6.1%)
- **Frameworks y librerías**:
  - ASP.NET Web Forms
  - Scripts personalizados en JavaScript
- **Base de datos**:
  - SQL Server / App_Data
